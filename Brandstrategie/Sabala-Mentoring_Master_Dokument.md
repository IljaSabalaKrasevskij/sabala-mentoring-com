## Technische Infrastruktur & Partnerschaft

### Technischer Stack

- Websites werden KI-basiert mit Antigravity (Google) und Claude Code erstellt
- HTML-Code wird in WordPress/Elementor implementiert durch Christopher
- Hosting auf deutschem Server — DSGVO-konform
- Keine personenbezogenen Daten auf US-Servern
- Antigravity arbeitet lokal auf dem Mac — keine Daten gehen raus
- Backup: GitHub + serverseitige Backups (täglich/monatlich)
- Cookie-Banner: Real Cookie Banner (schnell, beeinflusst Page Speed nicht)
- Rechtstexte: Automatisch über eRecht24, werden automatisch gepusht
- Performance: CSS/JavaScript wird asynchron und priorisiert geladen (Hero zuerst, Rest nachfolgend)
- Fonts: Google Fonts, aber lokal gespeichert (Datenschutz + Performance)

### Was der Kunde braucht

- Eine Domain (eigene oder neue)
- Webhosting (wir empfehlen einen Anbieter)
- Alles andere übernehmen wir

### Was der Kunde bekommt

- WordPress/Elementor-basierte Website — selbst Texte und Bilder änderbar
- Deutscher Server — DSGVO-konform
- Premium-Wartung — ein Mensch kümmert sich, nicht ein Bot
- Fortlaufende Updates, Sicherheits-Patches, Performance-Monitoring
- Erreichbarer Ansprechpartner bei technischen Fragen
- Serverseitige Backups
- Cookie-Banner und Rechtstexte automatisch aktuell

### Übergabe-Workflow (intern)

1. Ilja erstellt Website-Code in Antigravity
2. Pro Projekt: Ordner "für Christopher" mit HTML-Dateien, Bildern, Frames (WebP)
3. Christopher implementiert in WordPress/Elementor (ca. 1h pro Seite)
4. Christopher richtet Server ein: Domain, SSL, Backup, Cookie-Banner, Rechtstexte
5. Qualitätsprüfung: Performance, Responsiveness, Datenschutz
6. Go-Live
